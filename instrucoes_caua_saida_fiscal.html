Nova rota / página
URL: /guias/saida-definitiva-do-pais
Remix File-based routing

Copiar o arquivo para a pasta de rotas. A rota é criada automaticamente.

Destino do arquivo
app/routes/guias.saida-definitiva-do-pais.tsx
React Router v6 Router config centralizado

Copiar o arquivo para a pasta de páginas e adicionar a rota no config.

1 — Destino do arquivo
src/pages/GuiaSaidaDefinitiva.tsx
2 — Adicionar no router config
import GuiaSaidaDefinitiva from "./pages/GuiaSaidaDefinitiva";

// Dentro do array de routes — ANTES de qualquer rota dinâmica /guias/:slug:
{
  path: "/guias/saida-definitiva-do-pais",
  element: <GuiaSaidaDefinitiva />,
}
ℹ O arquivo usa import { Link } from "react-router-dom". Se o projeto usa @remix-run/react, trocar na linha 17 do arquivo.
⚠ O componente Accordion é do shadcn/ui. Se ainda não estiver instalado: npx shadcn@latest add accordion

Links internos — Jornadas
Adicionar links contextualizados nas jornadas relacionadas, se existirem
  • Jornada "Vou me mudar para o exterior" — sugerir leitura da Saída Fiscal antes de sair
  • Jornada "Moro no exterior" — oferecer caminho para regularização fiscal
  • Jornada "Estou voltando ao Brasil" — explicar reflexos do retorno na residência fiscal
Padrão de link contextualizado (mesmo estilo dos outros guias)
// Jornada: Vou me mudar
<Link to="/guias/saida-definitiva-do-pais" className="underline underline-offset-4 hover:text-foreground">
  Saída Fiscal — o que fazer antes de sair
</Link>

// Jornada: Moro no exterior
<Link to="/guias/saida-definitiva-do-pais" className="underline underline-offset-4 hover:text-foreground">
  Regularize sua situação fiscal no Brasil
</Link>

// Jornada: Voltando ao Brasil
<Link to="/guias/saida-definitiva-do-pais" className="underline underline-offset-4 hover:text-foreground">
  Saída Fiscal — entenda os reflexos do retorno
</Link>

Checklist de validação
Verificar antes de considerar a entrega concluída
  • Rota Rota /guias/saida-definitiva-do-pais abre a página corretamente
  • Rota H1 exibe "Comunicação de Saída Definitiva do País"
  • Nav Aba "Saída Fiscal" aparece no menu desktop
  • Nav Aba fica ativa (underline/estado) quando na URL da página
  • Guias Card 05 aparece corretamente em /guias
  • UI FAQ accordion abre e fecha corretamente
  • Links Links internos (CDT, outros guias) funcionam sem 404
  • Links Links externos (Receita Federal) abrem em nova aba
  • CTA Botões "Falar com especialista" apontam para /contato
  • Build Build e lint passam sem erros
  • Build Página responsiva e legível no mobile
